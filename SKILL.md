2. **数据真实性铁律**
   - 所有原始数据必须通过代码调用公开数据源获取，严禁编造数据；数据缺失需明确标注原因，不得臆测补全。

3. **逻辑严谨**：所有观点必须有对应数据与事实支撑，避免情绪化表达与无依据定性。
4. **口径统一约束**：所有财务、估值、行业指标必须明确标注统计口径（如归母/扣非、期末/平均、静态/TTM/动态、载重吨/艘数），全文同一指标口径必须完全一致，禁止混淆不同口径的数值。

## 分析框架

### 一、核心结论摘要（前置）
必须包含：
- **标的基础画像**：股票代码、市场、所属行业分类、主营业务核心构成、核心盈利模式、最新总市值
- **核心研究观点**：提炼3条以内核心逻辑（分优势逻辑与风险逻辑），明确当前研究判断（偏乐观/中性/偏谨慎）
- **强制标注**：> ⚠️ 本结论仅为研究参考，不构成投资建议。股市有风险，投资需谨慎。

### 二、行业与景气度分析
1. 所属行业当前生命周期阶段与景气度位置（上行/持平/下行）
2. **近期景气度边际变化**（强周期行业必填）：对比最新一周/一月现货价格与季度均值，说明当前处于加速上行、高位震荡还是见顶回落阶段
3. 近半年行业核心政策、上下游供需变化、技术迭代等关键变量；近90天重大事件优先分析
4. 公司在行业内的竞争格局：市占率排名、核心壁垒、对标龙头企业

【模块校验要点】
- 供需分析必须双侧量化：既讲供给约束也要讲在手订单/交付节奏，既讲需求增量也要讲需求下行风险；
- 市占率、运力占比等指标必须注明统计口径与数据来源；
- 禁止使用不存在的机构、券商名称，引用券商观点必须对应真实机构。

### 三、基本面与财务质量深度拆解

#### 3.1 商业模式深度拆解
从生意本质出发拆解盈利逻辑，所有定性判断必须结合后续财务数据交叉验证。
1. **盈利模式核心**：收入确认方式、结算方式、最小盈利单元模型、定价权来源
2. **成本结构与经营杠杆**：固定/变动成本占比、经营杠杆系数量化测算、核心成本驱动因子
3. **产业链地位与议价能力**：上下游话语权、应收/应付/预收验证资金占用能力、成本传导能力
4. **核心竞争壁垒（护城河）**：壁垒类型+量化佐证+强弱趋势判断
5. **扩张逻辑与增长模式**：增长驱动力拆解、扩张模式（轻/重资产）、资本开支匹配度
6. **现金流属性**：天然现金流特征、盈利向现金流的转化效率预判

#### 3.2 业务结构
- 近1年营收构成、毛利构成（分产品/分地区），以表格呈现
- 核心业务的增速变化与驱动因素

【数据规则】
- 若公司分部报告（分产品/分业务）的营收、利润绝对值未在公开接口中直接披露，禁止以“基于券商报告估算”为由凭空捏造数值。
- 合规呈现方式：
  1. 若年报/半年报附注中有分项数据，注明出处（如“根据2025年年报附注第X节”）；
  2. 若实时接口无法获取分项绝对值，仅列示已知的分项增速、占比区间，并注明“分项绝对值系依据上年同期公告推算，存在误差，仅供参考”；
- 严禁在无任何锚定数据（如上年分项绝对值、官方占比）的情况下，给出精确到“亿”的分项营收/利润预估。

#### 3.3 盈利能力分析（近3-5年趋势）
- 核心指标表格：ROE(摊薄)、ROIC、毛利率、净利率、扣非净利率
- ROE 杜邦分析（净利率 × 资产周转率 × 权益乘数）
- 同行对比：与行业均值/龙头的差异分析

【模块校验要点】
- 杜邦分析三因子必须采用统一口径（全部期末数，或全部平均余额），三因子乘积与ROE数值偏差不得超过0.5个百分点；
- 同行对比必须采用同一报告期、同一口径的指标。

#### 3.4 盈利质量与运营效率
- 净利润现金含量（经营现金流净额/净利润）
- 应收账款周转、存货周转、固定资产周转
- 三费费率变化趋势

#### 3.5 偿债与风险结构
- 资产负债率、有息负债率、货币资金/有息负债
- 流动比率、速动比率
- 商誉占净资产比例、股权质押比例

#### 3.6 成长能力
- 近3年营收、扣非净利润复合增速（CAGR）
- 单季度业绩增速环比/同比变化趋势

### 四、估值水平与盈利预期
1. **当前估值快照**：分口径列示静态PE、PE(TTM)、PB(LF)、PS(TTM)、EV/EBITDA
2. **估值分位**：当前估值在自身近3年/5年历史中的分位数
3. **行业对比**：与行业均值、可比公司的估值差异
4. **一致预期**：市场一致预期的未来2年净利润、对应动态PE；数据不足时按规则做谨慎估算
5. **简要估值测算**：选择1-2种适配方法，列明全部假设条件，给出理论估值区间，并注明"仅为理论测算，不代表价格预测"

【模块校验要点】
- 必须明确区分三类PE，命名不得混淆；
- 历史估值类比必须说明历史与当前的环境差异（如牛市/熊市、行业周期阶段），禁止直接用极端峰值作为合理参考；
- 估值区间计算过程必须可复现，数值必须准确。

### 五、量价与资金面分析
1. 近3个月/1年股价走势复盘，相对行业指数、大盘指数的超额收益
2. 技术面关键位置：重要均线状态、关键支撑压力区间
3. 资金面：北向资金/机构持仓变化、近1个月日均换手率与成交额
4. 筹码结构：股东人数变化、十大流通股东变动

【模块校验要点】
- "主力资金"类数据必须标注统计平台与定义，说明其局限性，不得作为核心论证依据；
- 股价涨跌幅、高低点数据全文必须统一，不得前后不一致。

### 六、风险因子排查
按影响程度从高到低逐项列明：
- 经营风险、财务风险、行业风险、市场风险、商业模式风险

【模块校验要点】
- 核心利好逻辑必须对应同等权重的风险分析，保持研究中立性；
- 禁止将低概率极端事件作为唯一风险点，必须覆盖周期波动、需求下滑等核心基本面风险。

### 七、后续跟踪要点
- 3-5个核心跟踪指标，附判断标准
- 财报披露、行业重要事件等关键时间节点

import akshare as ak
import pandas as pd
import numpy as np
from datetime import datetime, timedelta
import warnings
warnings.filterwarnings('ignore')
# Claude沙盒防超时配置
pd.set_option('display.max_rows', 20)
pd.set_option('display.max_columns', 10)

STOCK_CODE = "<用户输入的股票名称或代码>"
today = datetime.now().strftime("%Y%m%d")
start_3y = (datetime.now() - timedelta(days=3*365)).strftime("%Y%m%d")
start_1y = (datetime.now() - timedelta(days=365)).strftime("%Y%m%d")

# ========== 1. 基础信息与行情 ==========
stock_info = ak.stock_individual_info_em(symbol=STOCK_CODE)
daily_kline = ak.stock_zh_a_hist(
    symbol=STOCK_CODE, period="daily",
    start_date=start_1y, end_date=today, adjust="qfq"
)
latest_price = daily_kline['收盘'].iloc[-1]
# 总股本（万股）
total_share =  stock_info[stock_info['item']=='总股本']['value'].values[0]
# 市值校验：重新计算总市值（亿元）
total_mv_calc = latest_price * total_share / 10000
print(f"收盘价：{latest_price} 元 | 总股本：{total_share} 万股 | 计算总市值：{total_mv_calc:.2f} 亿元")

# ========== 2. 财务数据 ==========
fin_abstract = ak.stock_financial_abstract_ths(symbol=STOCK_CODE, indicator="按报告期")
latest_fin = fin_abstract.iloc[0]

# 核心指标提取
net_profit = latest_fin['净利润']  # 亿元
revenue = latest_fin['营业总收入']  # 亿元
total_assets = latest_fin['总资产']
net_assets = latest_fin['股东权益合计']
eps_annual = latest_fin['每股收益']
roe_reported = latest_fin['净资产收益率-摊薄']

# ========== 3. 核心指标计算与校验 ==========
# 3.1 分口径PE计算
pe_static = latest_price / eps_annual  # 静态PE
# TTM EPS需拼接过去4个季度，此处保留计算逻辑
# eps_ttm = 过去四季度净利润之和 / 总股本
# pe_ttm = latest_price / eps_ttm

# 3.2 ROE逆向校验 + 杜邦分解（统一期末口径）
net_margin = net_profit / revenue
asset_turnover = revenue / total_assets
equity_multiplier = total_assets / net_assets
roe_calc = net_margin * asset_turnover * equity_multiplier
print(f"披露ROE：{roe_reported} | 计算ROE：{roe_calc:.4%} | 偏差：{abs(roe_calc - float(roe_reported.strip('%'))/100):.4%}")

# 3.3 估值区间测算
expected_profit = 150  # 一致预期净利润，亿元，不足则用线性外推并标注假设
pe_low, pe_high = 8, 14
mv_low = expected_profit * pe_low
mv_high = expected_profit * pe_high
price_low = mv_low * 10000 / total_share
price_high = mv_high * 10000 / total_share
print(f"理论估值区间：{price_low:.2f} - {price_high:.2f} 元")
